## Summary

`beads` (`bd`) is a lightweight, graph-based issue tracker built for AI
coding agents to use as persistent memory across sessions. Installed via
the npm package `@beads/bd` (not the unrelated `beads` package, which is an
old, unrelated middleware-generator library — confirmed by inspecting its
`package.json` after a mistaken install). `bd init` was run in this repo,
which wired up Claude Code/Codex hooks and appended an integration block to
`CLAUDE.md`.
